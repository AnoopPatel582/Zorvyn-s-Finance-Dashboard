# Finance Dashboard UI

## Overview
A responsive finance dashboard to track transactions, analyze spending, and manage financial data.

## Features
- Summary cards (balance, income, expenses)
- Interactive charts (Recharts)
- Transactions table with search & filter
- Role-based UI (Admin / Viewer)
- Add transaction (Admin only)
- Insights (category + monthly comparison)
- LocalStorage persistence

## Tech Stack
- React
- Tailwind CSS
- Recharts

## Setup
```
npm install
npm run dev
```

## Approach
- Used Context API for state management
- Component-based architecture
- Focused on clean UI and usability
